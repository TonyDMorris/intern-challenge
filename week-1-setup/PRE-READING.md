# Week 1 → Week 2 Pre-Reading

A bit of React prep before you walk into the Week 2 session. You don't need to understand every line — aim to recognise the words "component", "props" and "state" when Eman says them.

## Watch (pick one)

- [React in 100 Seconds — Fireship](https://www.youtube.com/watch?v=Tn6-PIqc4UM) (2 min overview of what React is and why)
- [Learn React in 30 Minutes — Web Dev Simplified](https://www.youtube.com/watch?v=hQAHSlTtcmY) (a proper intro if you have the time)

## Read (skim, don't memorise)

- [React docs — "Your first component"](https://react.dev/learn/your-first-component)
- [React docs — "Passing props to a component"](https://react.dev/learn/passing-props-to-a-component)
- [React docs — "State: a component's memory"](https://react.dev/learn/state-a-components-memory)

## What you should be able to answer before Week 2

- What is a React component? (hint: a function that returns JSX)
- What are props? (hint: data passed from a parent to a child)
- What is state? (hint: data inside a component that can change over time)

If any of those feel fuzzy, that's fine — the Week 2 session starts with a recap.
