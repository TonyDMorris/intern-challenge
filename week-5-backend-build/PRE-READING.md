# Week 5 → Week 6 Pre-Reading

Week 6 is about building the React frontend for your game and wiring it to the Go backend you built this week. Quick refresher on fetch and useEffect before the session.

## Watch

- [useEffect explained — Web Dev Simplified](https://www.youtube.com/watch?v=0ZJgIjIuY7U) (how React loads data when a component appears)

## Read

- [React docs — Synchronizing with Effects](https://react.dev/learn/synchronizing-with-effects) (scroll to "Fetching data with Effects")
- [MDN — Using the Fetch API](https://developer.mozilla.org/en-US/docs/Web/API/Fetch_API/Using_Fetch) (skim the POST section — you'll need it)
- [`week-6-frontend-build/react-template/README.md`](../week-6-frontend-build/react-template/README.md) (5 min — skim what the template already provides so you know what you'll be changing)

## What you should be able to answer before Week 6

- What does `useEffect(() => { ... }, [])` do?
- How do you send JSON in a fetch POST?
- What file in the React template is *already filled in* so you don't have to rewrite it?
