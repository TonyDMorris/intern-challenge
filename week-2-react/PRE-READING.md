# Week 2 → Week 3 Pre-Reading

Before Week 3 you'll be connecting your React app to a Go backend. You don't need to know Go syntax in detail — just the shape of what a backend does.

## Watch

- [What is a REST API? — Fireship](https://www.youtube.com/watch?v=-MTSQjw5DrM) (what APIs are, how frontends and backends talk)
- [Go in 100 Seconds — Fireship](https://www.youtube.com/watch?v=446E-r0rXHI) (what Go is and why we use it)

## Read (skim)

- [A Tour of Go — Hello, World](https://go.dev/tour/welcome/1) (first 3-4 pages of the tour is enough)
- [MDN — Introduction to the server side](https://developer.mozilla.org/en-US/docs/Learn/Server-side/First_steps/Introduction)
- [MDN — Using the Fetch API](https://developer.mozilla.org/en-US/docs/Web/API/Fetch_API/Using_Fetch) (first few paragraphs — this is how React asks a backend for data)

## What you should be able to answer before Week 3

- What's the difference between a frontend and a backend?
- What does `fetch()` do in the browser?
- What's a JSON response?

Tony will recap all of this at the start of Week 3 — the reading is just so it's not brand new.
